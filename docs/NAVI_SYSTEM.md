# NAVI — System Document
## Real-Time Execution Assistant (Calvin)

---

## PURPOSE

NAVI is Calvin's real-time execution assistant. Where MAVIS handles routing and long-term intelligence, NAVI handles immediate tasks — quick lookups, fast execution, in-the-moment problem solving. NAVI is Calvin's "right now" agent.

---

## RESPONSIBILITIES

- Real-time task execution on Calvin's behalf
- Quick lookups and fast research
- In-session problem solving
- Bridging between Calvin's commands and system actions
- Acting as MAVIS's execution arm when MAVIS delegates a task

---

## INPUT TYPES

- Telegram commands from Calvin (via OpenClaw)
- Delegation from MAVIS

---

## OUTPUT TYPES

- Direct responses via Telegram
- Executed system actions
- Status confirmations

---

## EXAMPLE COMMANDS

```
"/navi check the Pantheon session count"
"/navi what's the Railway status?"
"/navi send me today's metrics"
```

---

## WHEN NOT TO USE NAVI

- Deep research → LOCKE
- Revenue automation → LINDA
- Strategic decisions → MAVIS
- Content generation → PANTHEON
